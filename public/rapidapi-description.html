TextKit API — AI Text Processing Toolkit

Transform any text with 9 production-ready AI endpoints. One API key, instant results.

Endpoints

Summarize Condense long text into key points and a concise summary
Rewrite Rewrite text in a different tone (formal, casual, persuasive, etc.)
Repurpose Convert content into platform-ready posts for Twitter, LinkedIn, etc.
SEO Meta Generate meta descriptions, title tags, and Open Graph data
Email Subject Lines Create high-converting email subject lines with style variants
Headlines Generate multiple headline options (listicle, article, clickbait)
Extract Keywords Pull keywords, named entities, and topics from any text
Tone Translate Detect the tone of text and translate it to a target tone
Compare Analyze similarity and differences between two texts

Why TextKit?

  • Simple JSON in, JSON out — no SDKs needed, works with any language
  • Structured responses — every endpoint returns clean, typed JSON (not raw LLM output)
  • Fast — median response time under 3 seconds
  • Rate limits by plan — scale from 100 to unlimited requests per day

Quick Start

curl -X POST "https://textkit-api.p.rapidapi.com/api/v1/summarize" \
  -H "X-RapidAPI-Key: YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"text": "Your long article text here..."}'
